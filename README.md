# Bank Management System

A console-based Bank Management System developed in C++ that simulates core banking operations using Object-Oriented Programming (OOP) and File Handling. The application allows users to create accounts, deposit money, withdraw funds, check balances, and manage customer records with persistent storage.

## 📌 Features

- Create New Bank Accounts
- Deposit Money
- Withdraw Money
- Check Account Balance
- View Account Details
- File-Based Data Storage
- Menu-Driven Interface
- Persistent Customer Record Management

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Standard Library

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/saicharanmareddy0110/Bank-Management-Application.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Bank-Management-System
   ```

3. Compile the program:
   ```bash
   g++ main.cpp -o bank
   ```

4. Run the executable:
   ```bash
   ./bank
   ```
